# Offline QLoRA training deps — INSTALL ONLY ON THE TRAINING MACHINE.
# These are intentionally NOT in requirements.txt: they are large (multi-GB),
# need a CUDA GPU, and must never load into the running browser engine. The engine
# imports nothing from here; only `src/training/finetune.py` (run as a script) does.
#
#   pip install -r requirements-training.txt
#
# A CUDA GPU is required for 4-bit QLoRA on a 7B model. Install the torch build
# that matches your CUDA (see https://pytorch.org); bitsandbytes needs CUDA too.
#
# Versions below are a known-compatible QLoRA stack. TRL's SFTConfig/SFTTrainer
# API drifts between releases — if finetune.py errors on kwargs, pin to these.

transformers>=4.44,<4.46
datasets>=2.20
peft>=0.12,<0.14
